# Top Down RPG

This is a simple top down RPG game made with Godot 3.2.3. 
The game is a simple prototype and is not meant to be a full game.

Engine: Godot 3.2.3

## How to run the game

1. Clone the repository
2. Open the project in Godot
3. Run the game
